# 📊 HR Analytics Dashboard Using Power BI

## 📌 Project Overview
This project presents an interactive **HR Analytics Dashboard** developed using **Power BI** to analyze employee data and generate meaningful insights. It helps organizations understand workforce trends and make data-driven decisions.

---

## 🚀 Features
- Employee Attrition Analysis
- Job Satisfaction Insights
- Department-wise Employee Distribution
- Performance Analysis
- Gender & Demographic Breakdown
- Interactive Filters & KPIs

---

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- SQL (for data handling)
- CSV Data Files

---

## 📂 Project Files
- `HR Analytics Dashboard Power BI.pbix` – Main dashboard file  
- `HR DATA_Excel.xlsx` – Raw dataset  
- `HR Analytics Dashboard Excel Project.xlsx` – Processed data  
- `hrdata.csv` – CSV dataset  
- `SQL Portfolio Project.docx` – Documentation  

---

## 📈 Key Insights
- Identifies high attrition departments  
- Analyzes employee satisfaction levels  
- Highlights workforce trends and patterns  
- Helps in strategic HR decision-making
