HR Analytics Project

Data Analytics using Excel, Power BI & SQL

📌 Objective

To analyze HR data and identify employee attrition trends, performance patterns, and workforce insights.

📊 Dataset

HR dataset containing employee details like age, salary, department, job role, attrition, etc.

🛠 Tools Used

  • Excel
  • Power BI
  • SQL
  • CSV Data

⚙️ Process

  • Data Cleaning using Excel
  • Data Analysis using SQL queries
  • Dashboard creation in Power BI
  • Visualization of HR metrics

📈 Key Insights

  • Attrition rate higher in certain departments
  • Low salary employees tend to leave more
  • Work-life balance affects retention

✅ Conclusion

This project helps HR teams make better decisions by understanding employee behavior and improving retention strategies.
